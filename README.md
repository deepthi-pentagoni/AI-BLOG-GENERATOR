# AI-BLOG-GENERATOR

AI Blog Generator is a Streamlit-based web application that generates well-structured blog posts from a user-provided topic using Generative AI. It provides a simple interface for creating high-quality content quickly and efficiently.

## Live Demo

https://ai-blog-generator-7.streamlit.app/

## Features

- Generate blog posts from a topic or prompt
- AI-powered content generation
- Clean and responsive Streamlit interface
- Fast and easy to use

## Tech Stack

- Python
- Streamlit
- Generative AI API (Gemini/OpenAI)
- Prompt Engineering

## Installation

```bash
git clone https://github.com/deepthi-pentagoni/ai-blog-generator.git
cd ai-blog-generator
pip install -r requirements.txt
streamlit run app.py
```
